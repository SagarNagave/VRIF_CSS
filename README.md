
•	 CSS HOME
CSS is used to style and design web pages.
It controls colors, layouts, fonts, and spacing.
•	CSS Introduction
CSS stands for Cascading Style Sheets.
It is used to make HTML pages look attractive.
•	CSS Syntax
CSS syntax consists of a selector and declaration block.
Example: h1 { color: blue; }.
 CSS Selectors
Selectors target HTML elements to apply styles.
Examples include element, class, and id selectors.
 CSS How To
CSS can be added using inline, internal, or external methods.
External CSS is the most commonly used.
 CSS Comments
Comments are notes in CSS that are not displayed.
They are written using /* comment */.
 CSS Errors
CSS errors happen due to wrong syntax or spelling.
They can stop styles from working properly.
 CSS Colors
CSS colors change text, background, and border colors.
They can be written using names, HEX, RGB, or HSL.
 CSS Backgrounds
Background properties control background color or images.
They help design the page appearance.
•	CSS Borders
Borders define lines around elements.
You can set style, width, and color.
•	CSS Margins
Margins create space outside an element.
They control distance between elements.
•	 CSS Padding
Padding adds space inside an element.
It increases space between content and border.
•	CSS Height / Width
Height and width set the size of elements.
Values can be in px, %, or other units.
•	CSS Box Model
The box model includes margin, border, padding, and content.
It controls spacing and layout structure.
•	CSS Outline
Outline is a line drawn outside the border.
It does not affect layout size.
•	CSS Text
Text properties style alignment, spacing, and decoration.
Example: text-align, text-transform.
•	CSS Fonts
Fonts control text style, size, and family.
They improve readability and design.
•	CSS Icons
Icons are small graphics added using CSS libraries or fonts.
They enhance UI design.
•	CSS Links
CSS styles link colors and hover effects.
It changes how links look and behave.
•	CSS Lists
CSS styles list bullets, numbers, and spacing.
It helps customize list appearance.
•	CSS Tables
CSS improves table layout and design.
It controls borders, spacing, and alignment.
•	CSS Display
Display property controls how elements appear.
Examples: block, inline, none.
•	CSS Max-width
Max-width sets the maximum width an element can have.
It helps make responsive layouts.
•	CSS Position
Position defines element placement on a page.
Types include static, relative, absolute, and fixed.
•	CSS Position Offsets
Offsets like top, left, right, bottom move positioned elements.
They work with position property.
•	CSS Z-index
Z-index controls element stacking order.
Higher value appears on top.
•	CSS Overflow
Overflow controls content that exceeds container size.
Options include hidden, scroll, or auto.

•	CSS Float
Float is used to position elements left or right inside a container.
It helps create layouts like text wrapping around images.
•	CSS Inline-block
Inline-block allows elements to stay inline but accept width and height.
It combines features of inline and block elements.
•	CSS Align
Align properties control horizontal or vertical positioning.
Used to center or adjust element placement.
•	CSS Combinators
Combinators select elements based on their relationship.
Examples include descendant, child, and sibling selectors.
•	CSS Pseudo-classes
Pseudo-classes style elements in special states.
Examples: :hover, :active, :focus.
•	CSS Pseudo-elements
Pseudo-elements style specific parts of elements.
Examples: ::before, ::after, ::first-letter.
•	CSS Opacity
Opacity controls the transparency of an element.
Values range from 0 (invisible) to 1 (full).
•	CSS Navigation Bars
Navigation bars are menus styled using CSS.
They help users move between pages easily.
•	CSS Dropdowns
Dropdown menus show hidden options when hovered or clicked.
They improve navigation design.
•	CSS Image Gallery
Image gallery layouts arrange images in rows or grids.
CSS controls spacing and alignment.
•	CSS Image Sprites
Image sprites combine multiple images into one file.
They improve website loading speed.
•	CSS Attribute Selectors
Attribute selectors target elements based on attributes.
Example: input[type="text"].
•	CSS Forms
CSS forms style input fields, labels, and buttons.
They improve the appearance of user input areas.
•	CSS Counters
Counters automatically number elements using CSS.
Useful for headings or lists.
•	CSS Units
Units define measurement values like px, %, em, and rem.
They control size and spacing.
•	CSS Inheritance
Inheritance means child elements get styles from parent elements.
It reduces repeated styling.
•	CSS Specificity
Specificity decides which CSS rule is applied.
More specific selectors override general ones.
•	CSS !important
!important forces a style to have highest priority.
It overrides other CSS rules.
 CSS Math Functions
Math functions perform calculations in CSS.
Example: calc() for dynamic sizes.
•	CSS Optimization
Optimization improves CSS performance and loading speed.
It includes minimizing and organizing code.
•	CSS Accessibility
Accessibility makes websites usable for all users.
CSS helps with contrast, focus styles, and readable layouts.
•	CSS Website Layout
Website layout arranges page structure using CSS.
Common methods include flexbox, grid, and positioning.
•	CSS Rounded Corners
Rounded corners make element edges curved using border-radius.
It gives a smooth and modern design look.
•	CSS Border Images
Border images use an image instead of a normal border.
It allows decorative borders around elements.
•	CSS Backgrounds
Background properties control color, images, and position.
They improve visual design of elements.
•	CSS Colors
CSS colors define text, border, and background colors.
They can be written using names, HEX, RGB, or HSL.
•	CSS Gradients
Gradients create smooth color transitions without images.
Examples include linear and radial gradients.
•	CSS Shadows
Shadows add depth using box-shadow or text-shadow.
They create a 3D-like visual effect.
•	CSS Text Effects
Text effects change appearance like spacing or overflow.
Used for styling and improving readability.
•	CSS Custom Fonts
Custom fonts allow using external fonts in a webpage.
They are added using @font-face.
•	CSS 2D Transforms
2D transforms move, rotate, scale, or skew elements.
They work in X and Y directions.
•	CSS 3D Transforms
3D transforms create depth effects using Z-axis.
They make elements look more realistic.
•	CSS Transitions
Transitions create smooth changes between styles.
They animate properties over time.
•	CSS Animations
Animations move or change elements automatically.
Created using @keyframes.
•	CSS Tooltips
Tooltips show small text when hovering over elements.
They provide extra information to users.
•	CSS Image Styling
Image styling controls borders, shadows, and size.
It enhances image appearance.
•	CSS Image Modal
Image modal displays images in a popup view.
Useful for galleries and previews.
•	CSS Image Centering
Image centering aligns images horizontally or vertically.
Done using margins or flexbox.
•	CSS Image Filters
Filters apply visual effects like blur or brightness.
Used with the filter property.
•	CSS Image Shapes
Image shapes change how images appear (circle, rounded).
Done using border-radius or clip-path.
•	CSS object-fit
Object-fit controls how images fit inside containers.
Examples include cover, contain, or fill.
•	CSS object-position
Object-position sets image alignment inside a container.
Works with object-fit property.
•	CSS Masking
Masking hides parts of an element using shapes or images.
It creates special visual effects.
•	CSS Buttons
CSS buttons style clickable elements.
They improve design and user interaction.
•	CSS Pagination
Pagination divides content into multiple pages.
Commonly used for blogs and e-commerce.
•	CSS Multiple Columns
Multiple columns split text into column layout.
Used for newspaper-style design.
•	CSS User Interface
UI properties improve usability and interaction.
Examples include cursor and resize.
•	CSS Variables
CSS variables store reusable values like colors or sizes.
Defined using --variable-name.




•	CSS @property
The @property rule defines custom CSS properties with specific types.
It helps control animation and variable behavior.
•	CSS Box Sizing
Box-sizing controls how width and height are calculated.
border-box includes padding and border inside size.
•	CSS Media Queries
Media queries apply styles based on screen size or device type.
They help create responsive web design

•	Flexbox Intro
Flexbox is a layout system for arranging items in rows or columns.
It makes alignment and spacing easier.
•	Flex Container
The flex container is the parent element with display:flex.
It controls layout of all child items.
•	Flex Items
Flex items are elements inside a flex container.
They can grow, shrink, or align automatically.
•	Flex Responsive
Flexbox helps create responsive layouts easily.
Items adjust based on screen size.

•	Grid Intro
CSS Grid is a two-dimensional layout system.
It arranges content in rows and columns.
•	Grid Container
The grid container uses display:grid.
It defines the main grid structure.
•	Grid Items
Grid items are elements placed inside the grid layout.
They can span rows or columns.
•	Grid 12-column Layout
A 12-column grid divides the page into equal sections.
It is commonly used for responsive designs.
•	CSS @supports
@supports checks if a browser supports a CSS feature.
It applies styles only when supported.

•	RWD Intro
Responsive Web Design makes websites adjust to all devices.
It improves usability on mobile and desktop.
•	RWD Viewport
Viewport controls how pages scale on different screens.
Used with the <meta viewport> tag.
•	RWD Grid View
Grid view divides layout into flexible columns.
It helps structure responsive pages.
•	RWD Media Queries
Media queries change styles for different screen sizes.
They are key for responsive design.
•	RWD Images
Responsive images adjust size automatically.
They prevent overflow on small screens.
•	RWD Videos
Responsive videos scale based on container size.
They maintain proper aspect ratio.
•	RWD Frameworks
Frameworks like Bootstrap help build responsive sites faster.
They provide ready layouts and components.
•	RWD Templates
Responsive templates are pre-designed layouts.
They adapt automatically to screen sizes.
